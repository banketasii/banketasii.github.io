RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method Link to EJS Link to JS
Index /guitars GET List all guitars Guitar.find() INDEX.EJS APP.JS
New /guitars/new GET Show new guitar form N/A NEW.EJS APP.JS
Create /guitars POST Create a new guitar, then redirect Guitar.create() No EJS File APP.JS
Show /guitars/:id GET Show info about a guitar Guitar.findById() SHOW.EJS APP.JS
Edit /guitars/:id/edit GET Show edit form for a guitar Guitar.findById() EDIT.EJS APP.JS
Update /guitars/:id PUT Update a guitar, then redirect Guitar.findByIdAndUpdate() No EJS File APP.JS
Destroy /guitars/:id DELETE Delete a guitar, then redirect Guitar.findByIdAndRemove() No EJS File APP.JS
